# Manual de Usuario
# NextEvent

## 1. Introducción

NextEvent es una aplicación web orientada a la gestión y organización de eventos. 
Permite a los usuarios crear eventos, administrar invitados, gestionar cronogramas, seleccionar locales, almacenar recuerdos mediante galerías y realizar evaluaciones del servicio.

Este manual describe el funcionamiento principal de la aplicación y está dirigido a los usuarios finales del sistema.

---

# 2. Acceso al Sistema

## Inicio de Sesión

Para ingresar al sistema el usuario debe acceder a la pantalla Login.

El usuario debe ingresar:

- Correo electrónico.
- Contraseña.

Luego debe presionar el botón de inicio de sesión.

Si los datos son correctos, el sistema permitirá el acceso al Dashboard principal.

---

# 3. Dashboard

El Dashboard representa la pantalla principal del sistema.

Desde esta sección el usuario puede acceder a las funcionalidades principales:

- Crear eventos.
- Administrar eventos existentes.
- Revisar cronogramas.
- Gestionar invitados.
- Acceder a locales.
- Revisar galerías.
- Configurar preferencias.

El objetivo del Dashboard es centralizar la navegación dentro de la plataforma.

---

# 4. Crear Evento

Esta sección permite registrar nuevos eventos dentro del sistema.

El usuario debe ingresar información relacionada con:

- Nombre del evento.
- Fecha.
- Descripción.
- Información adicional requerida.

Luego de completar los datos, el usuario puede guardar el evento.

El sistema registra la información y permite administrarla posteriormente.

---

# 5. Cronograma IA

El módulo Cronograma IA permite generar una planificación organizada para el evento.

Actualmente utiliza un servicio simulado preparado para una futura integración con un sistema de Inteligencia Artificial real.

Permite organizar actividades y visualizar la estructura temporal del evento.

---

# 6. Catálogo de Locales

Esta sección permite consultar los locales disponibles para realizar eventos.

El usuario puede visualizar información de los espacios registrados y seleccionar alternativas según sus necesidades.

La funcionalidad facilita la organización del evento centralizando las opciones disponibles.

---

# 7. Gestión RSVP

El módulo Gestión RSVP permite administrar la participación de invitados.

Funciones principales:

- Registrar invitados.
- Consultar estado de confirmación.
- Actualizar información de invitados.
- Organizar participantes del evento.

Los estados permiten conocer la confirmación de asistencia.

---

# 8. Galería

La sección Galería permite almacenar y visualizar contenido multimedia relacionado con los eventos.

El usuario puede consultar recuerdos asociados a sus eventos organizados.

---

# 9. Calificaciones

El módulo de Calificaciones permite registrar opiniones y valoraciones sobre los servicios utilizados.

Esta funcionalidad permite recopilar retroalimentación para mejorar la experiencia del usuario.

---

# 10. Configuración

La sección Configuración permite administrar preferencias generales del usuario.

Desde esta sección se pueden modificar opciones relacionadas con la cuenta y comportamiento del sistema.

---

# 11. Recomendaciones de Uso

Para utilizar correctamente la aplicación se recomienda:

- Mantener actualizados los datos registrados.
- Verificar la información antes de guardar eventos.
- Utilizar contraseñas seguras.
- Revisar periódicamente la información de invitados.

---

# 12. Conclusiones

NextEvent proporciona una plataforma integrada para la planificación y administración de eventos.

La aplicación permite gestionar las principales actividades relacionadas con la organización de eventos mediante una interfaz sencilla y módulos independientes.
